ClassDesk 사용 설명서

ClassDesk란?

영어 리딩 수업·공부방 운영자를 위한 올인원 수업 관리 도구입니다. 학생·수업·수업료·커리큘럼·교재·원서 라이브러리를 한 곳에서 관리하고, 원서에는 도서관 스타일의 레벨 라벨을 직접 인쇄해 붙일 수 있습니다.

  • 설치 불필요 — HTML 파일 하나를 브라우저(Chrome 권장)로 열면 끝입니다.
  • 내 컴퓨터에만 저장 — 모든 데이터는 브라우저 안(localStorage)에 저장됩니다. 서버 전송이 없어 학생 정보가 외부로 나가지 않습니다.
  • 자동화 — 책 정보 자동 입력, 학부모 알림장 자동 생성, 레벨 색상 자동 계산 등 반복 작업을 줄였습니다.

Books 자세히 — 원서 라이브러리

책 추가: 상단 검색창에 제목 입력 → Search → 결과 선택. 표지·저자·출판사·ISBN이 자동 입력되고, AR/Lexile 검색이 백그라운드로 실행됩니다. 저자명이 대문자(MARC BROWN)로 오면 자동으로 Marc Brown 형태로 정리됩니다.

  • Category / Genre — 형태(그림책·챕터북 등)와 주제(Animals·Adventure 등)를 드롭다운으로 선택. 목록에 없으면 "+ Custom…"으로 직접 입력. Genre는 Google 분류에서 자동 매핑되며, 같은 저자·같은 시리즈 책이 이미 있으면 그 분류를 자동으로 물려받습니다.
  • Level 자동 계산 — AR Level을 입력하면 색상 레벨(L1~L8)이 자동 표시됩니다. 기준: Red L1 (AR 0.0–0.9) · Orange L2 (1.0–1.9) · Yellow L3 (2.0–2.4) · Green L4 (2.5–2.9) · Blue L5 (3.0–3.4) · Purple L6 (3.5–3.9) · Pink L7 (4.0–4.9) · Black L8 (5.0+)
  • Search AR level / Search Lexile — 자동 검색이 못 찾은 경우 클릭 한 번으로 arbookfind.com / 구글 검색이 열립니다.
  • Search Google Images — 더 깨끗한 표지 이미지가 필요할 때. 제목이 클립보드에 복사된 채 이미지 검색이 열리고, 찾은 이미지를 커버 박스에 붙여넣으면 됩니다.
  • 필터·검색 — 카테고리, 장르, AR 범위, 레벨, Lexile, 라벨 인쇄 상태로 필터링. 검색창은 제목·저자·출판사·ISBN·장르·메모를 모두 찾습니다.

라벨 인쇄 — 준비물과 순서

준비물 (글로리텍 라벨지 2종):

  • SC9178 — 표지 라벨용. 30×21mm, A4 한 장에 78칸(6열×13행). 책 앞표지에 붙이는 미니 커버.
  • WP9140 — 정보 라벨용. 47×26.9mm, A4 한 장에 40칸(4열×10행). 색상 밴드(레벨 표시) + 제목 + 저자 + AR/Lexile + 카테고리 + 소장 표시(도서관 이름, Settings → Templates에서 변경).

인쇄 순서:

  1. Books에서 인쇄할 책들을 체크박스로 선택 → 상단 Covers 또는 Info 버튼 클릭
  2. 시작 칸 번호 입력 — 새 라벨지면 1, 쓰다 만 라벨지면 이어서 붙일 칸 번호. 칸 세는 방향: 표지 라벨은 세로(위→아래), 정보 라벨은 가로(왼쪽→오른쪽)입니다.
  3. 미리보기에서 배치 확인 (점선 = 라벨 칸 테두리, 인쇄에는 안 나옴) → Print
  4. 브라우저 인쇄 설정: 용지 A4 · Margins(여백) None · Scale 100% (Fit to page 끄기) · More settings에서 Background graphics 체크 (색상 밴드 인쇄용)
  5. 인쇄가 잘 나왔으면 ✓ Mark printed 클릭 → 목록에 인쇄 완료 표시가 저장됩니다

인쇄 추적: 목록의 C / I 뱃지가 초록이면 해당 라벨 인쇄 완료(마우스를 올리면 날짜 표시), 클릭하면 완료↔미완료 전환. 필터의 "Cover not printed"로 새로 추가한 책만 골라 인쇄할 수 있습니다. 테스트 인쇄는 Mark printed를 누르지 않으면 기록에 남지 않습니다.

급지 팁: 프린터마다 인쇄면 방향이 다릅니다. A4에 X를 그려 한 장 인쇄해 어느 면에 나오는지 확인한 뒤, 라벨 면을 그 방향으로 넣으세요. 라벨지의 위/아래 여백이 달라 넣는 위아래 방향도 항상 동일하게 유지해야 합니다.

학부모 알림장 & 결제 메시지

Settings → Templates에서 문구를 한 번 만들어두면, 수업 기록의 내용([student_name], [topic], [hw_page], [books_read] 등 플레이스홀더)이 자동으로 채워진 메시지가 생성됩니다. 복사해서 카톡 등으로 보내면 됩니다.

화면 커스터마이징

Settings → Custom Fields에서 각 페이지의 컬럼 순서(드래그), 이름 변경(클릭), 숨김(체크 해제)이 가능하고 나만의 필드도 추가할 수 있습니다. 숨긴 필드도 편집 창에는 항상 표시되므로 데이터 입력에는 지장이 없습니다.

자동 검색 설정 (API Keys) — 선택사항

책 정보 검색(Google Books)은 설정 없이 바로 작동합니다. 이미지 검색(🔍)과 AR/Lexile 자동 검색을 쓰려면 Settings → API Keys에 두 가지를 등록하세요:

  1. Google API 키 — Google Cloud Console에서 발급 후, "Custom Search API"를 사용 설정하고 키의 API 제한사항에 포함시키세요.
  2. Search Engine ID (cx) — programmablesearchengine.google.com에서 검색엔진을 만들면 발급됩니다.

등록 후 API Keys 탭의 테스트 버튼으로 연결을 확인하세요. 무료 할당량은 하루 100건입니다. 파일을 file:///로 직접 열면 키 제한 설정에 따라 차단될 수 있어요 — 아래 FAQ를 참고하세요.

데이터 보관 — 꼭 읽어주세요

  • 데이터는 이 브라우저에 저장됩니다. 같은 컴퓨터·같은 브라우저로 열면 파일을 새 버전으로 바꿔도 데이터가 유지됩니다.
  • 다른 컴퓨터로 옮기거나 브라우저를 바꿀 땐 Settings → Backup → Export로 파일을 내려받아 새 환경에서 Import 하세요.
  • 브라우저의 "인터넷 사용 기록 삭제"에서 사이트 데이터를 지우면 모든 데이터가 사라집니다. 정기적인 Export(백업)를 강력히 권장합니다.

자주 묻는 질문

인쇄

인쇄가 두 페이지로 나뉘어요
인쇄 설정에서 여백(Margins)을 None으로, 용지를 A4로, 배율을 100%로 확인하세요.
색상 밴드가 인쇄에 안 나와요
인쇄 창 More settings에서 Background graphics를 체크하세요.
라벨과 인쇄 위치가 몇 mm 어긋나요
프린터 급지 오차입니다. 급지 가이드를 용지에 딱 맞추고, 라벨지 넣는 방향(앞뒤·위아래)을 확인하세요. 위아래 여백이 달라 180° 돌려 넣으면 약 2mm 어긋납니다.
종이 뒷면에 인쇄돼요
급지 방향 문제입니다. A4에 X를 그려 테스트 인쇄해 어느 면에 나오는지 확인한 뒤, 라벨 면을 그 방향으로 넣으세요.
표지 라벨에 흰 테두리가 인쇄돼요
이미지 파일 자체에 포함된 스캔 여백입니다. 자동 제거가 시도되지만, 남는 경우 Search Google Images로 더 깨끗한 표지로 교체하세요.
Mark printed를 눌렀는데 반영이 안 돼요
인쇄 탭은 원래 ClassDesk 탭과 연결되어 있어야 합니다. ClassDesk 탭을 닫지 않은 상태에서 눌러주세요. 놓쳤다면 목록의 C/I 뱃지를 직접 클릭해 표시할 수 있습니다.

자동 검색 · API

AR/Lexile을 자동으로 못 찾아요
모든 책이 데이터베이스에 있지는 않습니다. Search AR level / Search Lexile 링크로 확인 후 직접 입력하세요.
arbookfind에 "AR Level"이 없고 "ATOS Book Level"만 있어요
같은 값입니다. ATOS는 AR 레벨의 공식 명칭이에요 — 그 숫자를 AR Level 칸에 입력하면 됩니다.
이미지 검색 버튼이 "Search failed (HTTP 403)" 오류를 내요
Google API 설정 문제입니다. ① Google Cloud Console에서 Custom Search API가 "사용 설정"되어 있는지, ② API 키의 "API 제한사항"에 Custom Search API가 포함되어 있는지 확인 후, Settings → API Keys에 API 키와 Search Engine ID(cx)를 모두 저장하세요.
"Requests from referer null are blocked" 오류가 나요
파일을 file:///로 직접 열었을 때 발생합니다. VSCode Live Server 등 로컬 서버(127.0.0.1)로 열거나, Google Cloud Console에서 API 키의 "애플리케이션 제한사항"을 "없음"으로 바꾸세요.
잘 되던 자동 검색이 갑자기 안 돼요 (HTTP 429)
Google Custom Search 무료 할당량(하루 100건)을 초과한 것입니다. 다음 날 자동으로 리셋됩니다. 대량 등록하는 날은 수동 입력을 섞어 쓰세요.
책 검색 결과에 원하는 책이 안 나와요
제목 + 저자로 검색해 보세요 (예: "arthur snow marc brown"). 한국 유통사 등록본은 제목에 "(CD포함)" 같은 문구가 붙어 오기도 하니, 불러온 뒤 제목을 정리하면 됩니다.

기타

날짜·요일 계산이 컴퓨터 시간과 달라요
ClassDesk의 "오늘"은 항상 한국 표준시(KST) 기준입니다. 해외에서 열거나 컴퓨터 시간대가 달라도 수업 스케줄은 한국 시간으로 계산됩니다.
편집 창에서 특정 입력칸이 안 보여요
편집 창에는 모든 필드가 항상 표시됩니다. Custom Fields의 숨김 설정은 목록 컬럼에만 적용돼요. 그래도 안 보이면 파일이 구버전인지 확인하세요.
파일을 새 버전으로 교체하면 데이터는요?
같은 컴퓨터·같은 브라우저라면 유지됩니다. 데이터는 파일이 아니라 브라우저에 저장되기 때문이에요. 안전을 위해 교체 전 Export를 권장합니다.
